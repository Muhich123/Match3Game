# AGENTS.md

## Unity project rules
- This is a Unity 2022.3.37f1 project.
- Do not delete original assets.
- Preserve .meta files.
- Do not edit gameplay logic unless explicitly requested.
- Do not modify monetization, ads, IAP, analytics, or external services unless explicitly requested.
- Put Halloween texture variants into Assets/PuzzleMatchKit/Resources/Game/Halloween/ if possible.
- If a folder does not exist, create it.
- Preserve texture resolution and import settings unless asked.
- If changing sprites/materials/prefabs, update only references needed for the visual style.
- Keep the original project working.
- After changes, update README.md with what changed and how to test it.

## Visual style request defaults
- Halloween style should be dark, cozy, readable, and game-friendly.
- Prefer pumpkins, candy, ghosts, skulls, candles, bats, purple/orange accents.
- Avoid over-darkening playable blocks so the player can still distinguish objects.
